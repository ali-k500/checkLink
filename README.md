# checkLink

## AI-Powered Phishing Detection Website

### About

checkLink is a simple website that helps people detect phishing links and scam messages using AI.

---

## Problem

Many people receive fake links and scam messages every day. It is not always easy to know if a link is safe or dangerous.

---

## Challenge

Build a simple tool that helps users check links before opening them.

---

## Target Users

* Students
* Families
* Teachers
* Employees
* Anyone who uses the Internet

---

## Solution

Users paste a suspicious link or message into the website.

The AI analyzes it and provides:

* A risk score (0–100)
* Safe, Suspicious, or Dangerous result
* A simple explanation
* Cybersecurity tips

---

## Features

* Check suspicious links
* Analyze scam messages
* AI-powered analysis
* Risk score
* Easy-to-understand results
* Cybersecurity awareness tips
* Arabic interface
* Mobile-friendly design

---

## How It Works

1. Paste a suspicious link or message.
2. Click **Analyze**.
3. View the result.
4. Read the explanation and safety tips.
